# 龙之核心

## ![介绍](_images/Dragon_Core/1.jpg)

龙之核心是基于 Minecraft 1.12.2 版本，以插件(服务端) + MOD(客户端) 的方式，
实现了丰富的在服务端制作各种客户端功能的方法。

与 [萌芽引擎](Germ_Engine.md) 为同类型插件
